# Sample Dataset Details

## Dataset Title
Student Academic Performance Dataset

## Dataset Description
This dataset contains information related to students and their academic performance.
It is created as a sample dataset for educational and demonstration purposes.

## Objective
To analyze student performance based on academic and demographic attributes.

## Attributes / Columns
1. Student_ID
2. Name
3. Age
4. Gender
5. Course
6. Semester
7. Marks
8. Attendance Percentage

## Data Type
- Student_ID: Integer
- Name: String
- Age: Integer
- Gender: String
- Course: String
- Semester: Integer
- Marks: Integer
- Attendance Percentage: Float

## Number of Records
100 student records

## Data Collection Method
The data is manually created for academic use.

## File Format
CSV

## Usage
This dataset can be used for basic data analysis, visualization, and learning purposes.
